# DatePickerPane

> 日期面板（无输入框）

适合在表单中嵌入内联面板或自定义弹层布局时使用。它拥有与 `DatePicker` 相同的 props，并会自动应用相同的 `format` / `valueFormat` 默认值，因此无需额外配置即可获得统一的字符串值。
