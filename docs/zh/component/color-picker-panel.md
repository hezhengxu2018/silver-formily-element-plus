# ColorPickerPanel

> 颜色选择面板（无输入框）

用于在表单中嵌入纯拾色面板或自定义弹层内容，拥有与 `ColorPicker` 相同的 props，默认遵循 `ColorPicker` 的格式化与禁用逻辑，因此在受控表单里不需要额外适配即可获取字符串值。
